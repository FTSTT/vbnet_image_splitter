长图分割标记工具

基于 VB.NET 的桌面工具，将整页长图按蓝线精准切分为多张图片。 支持蓝线标记、删除区域、插入空白、缩放导航、导出 PNG/JPG，以及工程保存与恢复 （包含完整动作历史重放与原始高度校验），适合处理漫画、教程长图、条漫截屏等。

Windows • VB.NET WinForms 蓝线分割 / 删除区域 插入空白 • 撤销重做 PNG/JPG 导出 • 工程恢复
主界面示意
在这里预览长图、添加蓝线、删除区域、插入空白并导出分块。顶部工具栏和导出选项支持快捷操作。

功能特性

围绕“长图按横向分割线切图”这一核心场景，工具提供完整的标线、编辑、导出与工程管理能力。

蓝线标记与分割

左键添加蓝线，默认保留顶部 Y=0 蓝线。右键单击可删除区域，右键拖拽可高亮并批量删除选区内蓝线。

区域删除与插入空白

右键删除蓝线之间区域，中键配合“插入空白高度”在任意位置插入纯白空白，所有动作可撤销/重做。

导出 PNG/JPG

支持导出 PNG 和 JPG，JPG 可配置质量值，支持全局统一宽度或按区域最大宽度导出。

Trim 与 Padding

可自动去除分块四边纯白边缘，并为左右上下分别增加白边，以像素为单位精细控制导出尺寸。

纯白分块检测

导出前检测纯白分块并提示是否跳过，避免生成无意义空白图片。

工程与历史重放

工程文件保存图片列表与完整动作历史（含插入空白），加载时从原始图片开始重放动作并校验原始高度。

使用说明（简要）

更多细节请参考 GitHub 仓库中的 README.md，这里只保留核心操作说明。

打开图片与工程

使用“打开文件夹”或“打开文件(&O)”加载图片，程序会自动按数字序号排序并拼接为长图。 使用“读取工程”可恢复上一次保存的蓝线、删除区域、插入空白以及导出设置。

分割线与区域编辑

顶部固定一条 Y=0 蓝线。左键添加蓝线；右键单击删除蓝线与当前位置之间的区域； 右键拖拽选中一段纵向区域并高亮其中蓝线，可选择是否批量删除这些蓝线。 中键单击按照“插入空白高度”在当前位置插入纯白空白。

导出与纯白分块处理

在导出选项中设置 Trim、Padding、宽度策略和导出格式。 导出时，如检测到纯白分块会提示是否导出这些分块。对已有文件将提前确认是否覆盖。
